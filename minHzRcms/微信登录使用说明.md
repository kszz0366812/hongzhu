# 微信登录功能使用说明

## 功能概述

本项目已完整实现了微信小程序的静默登录功能，用户无需手动操作即可自动完成登录。

## 实现特性

### 1. 静默登录
- 小程序启动时自动执行登录
- 无需用户授权即可获取 openid
- 登录状态持久化存储

### 2. 用户信息管理
- 支持用户主动获取微信用户信息
- 用户信息本地缓存
- 支持员工信息绑定

### 3. 错误处理
- 完善的错误码处理机制
- 网络请求重试机制
- 友好的错误提示

## 配置说明

### 1. 修改配置文件

编辑 `config/wx-config.js` 文件：

```javascript
const config = {
  // 替换为你的API地址
  API_BASE_URL: 'https://your-domain.com/api',
  
  WECHAT: {
    // 替换为你的小程序 appId
    APP_ID: 'your-app-id',
    APP_SECRET: 'your-app-secret'
  }
}
```

### 2. 后端API要求

后端需要实现以下API接口：

#### 微信登录接口
- **地址**: `POST /api/auth/wx/login`
- **参数**: 
  ```json
  {
    "code": "微信登录code",
    "userInfo": {
      "nickName": "用户昵称",
      "avatarUrl": "头像URL",
      "gender": 1,
      "country": "中国",
      "province": "广东",
      "city": "深圳"
    }
  }
  ```
- **返回**:
  ```json
  {
    "code": 200,
    "message": "success",
    "data": {
      "success": true,
      "token": "JWT token",
      "openid": "用户openid",
      "unionid": "用户unionid",
      "userInfo": {
        "id": 1,
        "nickname": "用户昵称",
        "avatarUrl": "头像URL",
        "gender": 1,
        "country": "中国",
        "province": "广东",
        "city": "深圳",
        "isBound": 0
      }
    }
  }
  ```

#### 更新用户信息接口
- **地址**: `POST /api/user/update`
- **参数**: 微信用户信息对象

#### 检查员工信息接口
- **地址**: `POST /api/auth/wx/check-employee`
- **参数**: `{ "phone": "工号" }`

#### 绑定员工信息接口
- **地址**: `POST /api/auth/wx/bind-employee`
- **参数**: `{"employeeCode": "工号" }`

## 使用方法

### 1. 自动登录

登录功能已在小程序启动时自动执行，无需手动调用。

### 2. 检查登录状态

```javascript
const auth = require('./utils/auth.js')

// 检查是否已登录
const isLoggedIn = auth.checkLoginStatus()

// 获取用户信息
const userInfo = auth.getUserInfo()

// 获取token
const token = auth.getToken()
```

### 3. 获取用户信息

```javascript
const app = getApp()

// 获取用户信息（需要用户授权）
app.getUserProfile()
  .then((userInfo) => {
    console.log('用户信息:', userInfo)
  })
  .catch((error) => {
    console.error('获取失败:', error)
  })
```

### 4. 员工绑定

```javascript
const auth = require('./utils/auth.js')

// 检查员工信息
auth.checkEmployeeBinding('13800138000')
  .then((response) => {
    if (response.data.exists) {
      // 绑定员工
      return auth.bindEmployee(response.data.employeeId, '13800138000')
    }
  })
  .then(() => {
    console.log('绑定成功')
  })
```

### 5. 退出登录

```javascript
const auth = require('./utils/auth.js')

auth.logout()
```

## 注意事项

1. **API地址配置**: 请确保在 `config/wx-config.js` 中配置正确的API地址
2. **小程序配置**: 确保小程序已正确配置 appId 和 appSecret
3. **网络请求**: 建议在生产环境中使用HTTPS协议
4. **错误处理**: 已实现完善的错误处理机制，包括网络重试
5. **存储安全**: 敏感信息已安全存储在本地存储中
