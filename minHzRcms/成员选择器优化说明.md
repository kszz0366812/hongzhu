# 成员选择器优化说明

## 优化内容

### 1. 模态框高度优化
- 将模态框最大高度从 `90vh` 增加到 `95vh`
- 使用 flexbox 布局，确保各个区域合理分配空间
- 搜索框和已选成员区域设置为 `flex-shrink: 0`，防止被压缩
- 成员列表区域设置为 `flex: 1`，占据剩余空间

### 2. 已选成员显示区域优化
- **固定显示**：已选成员区域在多选模式下始终显示，避免高度闪烁
- **固定高度**：设置 `min-height: 120rpx` 和 `max-height: 120rpx`，确保布局稳定
- **横向滚动**：使用 `scroll-view` 组件实现横向滚动，支持多个已选成员
- **空状态提示**：当没有选择成员时显示"请选择成员"提示
- **事件隔离**：添加 `catchtouchmove` 和 `catchscroll` 防止滚动事件冒泡

### 3. 成员列表头像显示
- 使用默认图标 `👤` 代替头像图片
- 统一头像样式，避免图片加载失败问题
- 头像大小设置为 `60rpx`，与文字对齐
- 已选成员头像大小为 `32rpx`

### 4. 样式优化
- 优化已选成员的布局和样式
- 改进成员列表的视觉效果
- 确保在不同屏幕尺寸下的适配性
- 优化滚动条和交互体验

## 技术实现

### 布局结构
```
member-selector-modal (flex column)
├── member-selector-search-section (flex-shrink: 0)
├── selected-members-section (flex-shrink: 0, 固定高度120rpx)
│   └── scroll-view (horizontal scroll, 固定高度60rpx)
│       └── selected-members-list
├── member-selector-list (flex: 1)
└── member-selector-footer (flex-shrink: 0)
```

### 关键样式
- 模态框高度：`max-height: 95vh`
- 已选成员区域：`min-height: 120rpx, max-height: 120rpx`
- 滚动区域高度：`height: 60rpx`
- 头像图标：`👤` (Unicode: U+1F464)
- 成员列表最小高度：`min-height: 400rpx`

### 事件处理
- 使用 `catchtouchmove` 和 `catchscroll` 防止滚动事件冒泡
- 改进 `stopPropagation` 方法，正确处理事件对象
- 确保滚动操作不会影响父级页面

## 修复的问题

### 1. 滚动条事件冒泡问题
- **问题**：滚动条响应到了下一级页面的事件
- **解决**：在 `scroll-view` 上添加 `catchtouchmove` 和 `catchscroll` 事件处理
- **效果**：滚动操作被正确隔离，不会影响父级页面

### 2. 已选人员区域高度闪烁问题
- **问题**：已选人员区域需要等选择了人员之后才弹出，高度会突然闪一下
- **解决**：已选人员区域在多选模式下始终显示，设置固定高度
- **效果**：布局稳定，无高度闪烁

### 3. 头像显示问题
- **问题**：头像未显示出来
- **解决**：使用默认图标 `👤` 代替头像图片
- **效果**：统一显示效果，避免图片加载失败

## 使用说明

### 基本用法
```xml
<member-selector 
  show="{{showSelector}}"
  title="选择成员"
  placeholder="搜索成员..."
  members="{{memberList}}"
  multiple="{{true}}"
  selectedIds="{{selectedIds}}"
  confirmText="确定"
  bind:close="onClose"
  bind:select="onSelect"
/>
```

### 成员数据格式
```javascript
{
  id: 1,
  name: '张经理',
  role: '项目经理'
  // 不再需要 avatar 字段，使用默认图标
}
```

## 注意事项

1. 多选模式下，已选成员区域始终显示，避免布局闪烁
2. 滚动事件被正确隔离，不会影响父级页面
3. 使用统一的人形图标 `👤` 作为头像
4. 搜索功能支持按名称搜索
5. 模态框高度会根据内容自动调整
6. 已选成员区域有固定高度，确保布局稳定
